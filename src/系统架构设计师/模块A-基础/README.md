# 模块 A：计算机与系统基础

> 预计分值：上午题 15-20 分
> 学习周期：第 1-4 周

本模块是整个考试的基础，涵盖计算机组成、操作系统、网络和数据库四大领域。

## 子模块

- [A1 计算机组成与体系结构](A1-计算机组成/README.md) — CPU、存储系统、流水线、可靠性 ⭐必考
- [A2 操作系统](A2-操作系统/README.md) — 进程管理、内存管理、死锁、PV 操作 ⭐必考
- [A3 计算机网络](A3-计算机网络/README.md) — TCP/IP、HTTP、安全协议
- [A4 数据库系统](A4-数据库/README.md) — SQL 优化、事务、分布式数据库

## 学习建议

1. 计算机组成和操作系统是历年必考重点，优先攻克
2. 计算机网络如果你有前端经验，大部分内容已经熟悉
3. 数据库需要重点复习 SQL 优化和分布式相关概念
